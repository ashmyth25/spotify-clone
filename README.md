# 🎵 Spotify Clone

A modern and responsive **Spotify-inspired web music player** built using **HTML, CSS, and JavaScript**. This project recreates the look and feel of Spotify's desktop interface with interactive features such as playlist navigation, music player controls, search, library, and responsive design.

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/ashmyth25/spotify-clone.git
```

Go into the project folder

```bash
cd spotify-clone
```

Open `index.html` in your browser.

Or use **Live Server** in VS Code.
